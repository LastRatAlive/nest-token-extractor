Nest Token Extractor

Extracts fields needed for Home Assistant integrations and Homebridge plugins. Works for both Google and Legacy Nest accounts.

⚠️ CRITICAL WARNING FOR GOOGLE CHROME: Google Chrome utilizes hardware-bound session security (DBSC) for Google Accounts. Credentials extracted here using Chrome will fail in Home Assistant either immediately or after a few hours. Please use Firefox or Safari instead.
💡 NOTE FOR FIREFOX USERS: If you encounter an empty Cookies field, ensure you have clicked the shield icon in the Firefox address bar on home.nest.com and turned off Enhanced Tracking Protection. This allows Google's session cookies to be securely extracted.
